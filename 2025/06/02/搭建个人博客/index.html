基于Hexo搭建个人博客，并托管于GitHub

发表于2025-06-02|更新于2025-06-02
|浏览量:

1、安装必备的node与git
此处需要留意，node的版本不要太低，低于12以下，会出现下面的报错。

!(C:\Users\赵祖银\my-blog\source_posts\image\image1.png)

其次，若电脑存在多个版本的node，可使用nvm（对中文目录很敏感）进行控制。
2、安装Hexo
在指定目录下，执行下面的命令

之后，就可在目录下写第一篇文章

3、本地运行Hexo
首次运行，需要执行以下命令
hexo clean
hexo g
hexo s
之后便可直接在本地进行访问

4、部署到gitHub

首先需要在github上创建一个仓库
其次，安装部署插件
npm install hexo-deployer-git –save
然后，修改config.yml文件中的deploy配置，并追加配置

这个地方不支持根据用户名和密码进行push，需要使用ssh的方式

最后，执行hexo d，push到github

5、部署完成之后
便可访问 https://你的用户名.github.io，github托管更新内容会不及时，过一分钟就好了。
6、最终效果

此处选择了Butterfly主题进行美化，官方文档https://butterfly.js.org/posts/21cfbf15/，可结合个人喜好自定义选择。
